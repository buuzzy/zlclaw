## 待实现 📋

### P1 — OKX 全链路交易集成

加密资产方向：行情 → 分析 → 下单完整闭环。

| 阶段 | Skill | 说明 |
|------|-------|------|
| 第一阶段 | `okx-market` | 行情与数据（只读） |
| 第二阶段 | `okx-account` | 账户与持仓（只读，需 API Key） |
| 第三阶段 | `okx-trade` | 下单执行（含确认卡片） |

核心原则：AI 只负责计算和提案，执行权始终在用户手中。
