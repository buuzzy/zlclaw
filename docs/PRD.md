# HTclaw — 产品需求文档（PRD）

> 版本：v0.1  
> 更新日期：2026-04-19  
> 状态：持续迭代中

---

## 一、产品定位

**HTclaw 是面向中国个人投资者的本地 AI 金融助手。**

核心价值主张：
- **本地优先**：数据和对话历史存储在用户本地（SQLite），不上传第三方服务器
- **垂直深度**：内置 A 股行情、投研分析、选股筛选等领域技能，开箱即用
- **渠道整合**：与飞书、微信无缝集成，在日常工作渠道中直接触发 AI 分析
- **主动触达**：Cron 调度 + 渠道推送，无人值守完成分析并主动通知用户

---

## 二、目标用户

### 主要用户：中国个人投资者

**特征：**
- 关注 A 股（沪深两市）+ 加密资产（BTC/ETH 等）
- 日常使用飞书/微信办公，希望在工作渠道直接获取市场信息
- 有一定投资经验，能理解技术指标和研报语言
- 对数据隐私敏感，不希望投资偏好和持仓信息上传云端

**核心痛点：**
- 信息分散：行情、研报、公告分布在多个 App 中，需要手动汇总
- 决策链路长：从看到信号到下单需要切换多个工具，执行效率低
- 信息噪声大：市场信息质量参差不齐，需要 AI 辅助筛选和解读
- 无记忆：现有 AI 工具不记得用户偏好，每次都要重新交代背景

### 次要用户：量化爱好者 / 私募从业者

**特征：**
- 需要批量处理行情数据、生成分析报告
- 有定制化脚本需求
- 可能同时管理 A 股和加密资产两个市场

## 三、核心功能模块

### 3.1 对话 Agent（主功能）

用户通过自然语言与 AI 交互，AI 调用对应技能完成任务。

**支持场景：**
- 行情查询：「茅台今天怎么样？」→ 调用 westock-market
- 投研分析：「分析一下宁德时代的基本面」→ 调用 westock-research
- 选股筛选：「找出 PE<20 且近三年 ROE>15% 的消费白马股」→ 调用 westock-screener
- 板块分析：「今天哪些板块资金流入最多？」→ 调用 westock-sector
- 定时任务：「每天早上 9 点给我发一个市场早报」→ 调用 Cron 技能

### 3.2 记忆系统

AI 记住用户的投资偏好、历史讨论和分析结论。

**功能：**
- 每轮对话自动记录到 `~/.htclaw/memory/YYYY-MM-DD.md`
- 每晚 23:00 自动将日记录提炼为结构化长期记忆（MEMORY.md）
- 新对话开始时自动注入相关历史上下文（向量 + FTS 混合检索）
- 用户可在 Settings > Memory 查看记忆状态、手动触发归纳

### 3.3 渠道集成

将 HTclaw 的 AI 能力延伸到飞书/微信等工作渠道。

**飞书功能：**
- 在飞书群或单聊中 @ 机器人，触发 AI 分析
- 返回富文本卡片（行情数据、K 线图、分析结论）
- 支持流式响应（打字效果）
- API 速率限制自动处理（指数退避重试）

**微信功能：**
- 微信公众号 / 企业微信接入
- 消息同步到 HTclaw 本地任务列表

### 3.4 定时调度（Cron）

无人值守的定时分析 + 主动推送。

**调度类型：**
- `cron`：标准 cron 表达式（如 `15 9 * * 1-5` 每工作日 9:15）
- `every`：固定间隔（如每 5 分钟检查一次价格）
- `at`：一次性任务（如明天上午 10 点提醒我）

**内置系统任务：**
- `sys-memory-consolidation`：每天 23:00 自动记忆归纳

### 3.5 本地数据存储

所有用户数据存储在本地，不上传。

**存储结构：**
```
~/.htclaw/
  ├── config.json          # API Key、模型配置
  ├── MEMORY.md            # 长期记忆（consolidator 输出）
  ├── user.md              # 用户画像
  ├── memory/              # 日对话记录
  │   └── YYYY-MM-DD.md
  ├── skills/              # 用户自定义技能
  ├── cron/
  │   └── jobs.json        # 定时任务持久化
  └── db/
      └── htclaw.db        # SQLite 主数据库
```

## 四、非功能需求

### 4.1 隐私与安全

- 所有用户数据本地存储，不经过 HTclaw 服务器
- API Key 仅存于 `~/.htclaw/config.json`，不随代码提交
- 管理类 API（`/memory/*`、`/cron/*`、`/settings/*`）通过 `localOnlyMiddleware` 限制只有本机访问

### 4.2 性能

- 应用冷启动时间 < 3 秒
- 向量检索响应 < 200ms（p95）
- 飞书卡片首次渲染 < 1 秒

### 4.3 可靠性

- Cron 任务持久化，应用重启后恢复调度
- 记忆归纳幂等（崩溃重启不重复处理）
- 渠道推送失败自动重试（最多 3 次，指数退避）

### 4.4 平台支持

- 主要平台：macOS（Apple Silicon + Intel）
- 次要平台：Windows（计划中）
- 打包形式：Tauri 原生安装包（DMG / MSI）

- 飞书 / 微信渠道集成
