# v1.0.0 — 首发版本

> 2026-04-22

涨乐金融龙虾桌面版首发。AI 驱动的金融研究助手。

## 核心能力

- **17 个内置金融技能**：行情、财务、基本资料、公司经营、行业、指数、基金、宏观、公告、新闻、研报、定时任务、web-access、westock-market、westock-quote、westock-research、westock-screener
- **13 个金融可视化组件**：QuoteCard、KLineChart、BarChart、LineChart、DataTable、NewsCard、StockSnapshot、SectorHeatmap、ResearchConsensus、FinancialHealth、NewsFeed、FinanceBreakfast、AIHotNews
- **记忆系统**：每日归纳、向量检索、三层保留策略
- **Cron 定时调度**：Agent 自然语言创建任务
- **多渠道推送**：飞书 / 微信
- **Local-first**：所有完整数据存本地，云端只同步元数据 / 偏好
- **OpenAI 兼容端点**：第三方客户端可直接接入

## 安装

下载 `涨乐金融龙虾_1.0.0_aarch64.dmg`（macOS Apple Silicon），拖入 Applications 即可。

Intel Mac 和 Windows 版本将在后续 release 中提供。

## 已知限制

- DMG 体积约 250MB（后续通过 skill 按需下载优化）
- 首次启动需在 Settings → AI & API 配置 API Key
